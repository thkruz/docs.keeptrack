The KeepTrack project is licensed under the GNU Affero General Public License (AGPL). The AGPL is a free, copyleft license that allows users to access and modify the source code. It also requires that any modifications to the software be released under the same license. The AGPL is designed to ensure that users can access and modify the software, even when it is deployed as a web service.

## License Summary

The AGPL includes the following key points:

- **Copyleft**: The AGPL is a copyleft license, which means that any modifications to the software must be released under the same license.
- **Source Code Availability**: The AGPL requires that the source code of the software be made available to users when the software is accessed over a network.
- **Distribution of Modifications**: Any modifications to the software must be distributed under the same license terms as the original software.
- **Commercial Use**: The AGPL allows for commercial use of the software, but requires that any modifications be released under the same license.

## License Compatibility

The AGPL is compatible with other copyleft licenses, such as the GNU General Public License (GPL), but may not be compatible with proprietary licenses. If you plan to use the KeepTrack project in combination with other software, it is important to consider the compatibility of the licenses.

## License Enforcement

The AGPL includes provisions for enforcing compliance with the license terms. If you believe that a user or organization is violating the terms of the AGPL, you can take legal action to enforce the license.

## License Violations

If you encounter a violation of the AGPL license terms, you can report the violation to the [Free Software Foundation](https://www.fsf.org/). The Free Software Foundation can provide guidance on how to address the violation and ensure that the software remains compliant with the AGPL.
