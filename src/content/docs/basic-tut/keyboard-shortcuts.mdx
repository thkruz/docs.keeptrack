The keyboard shortcuts are based on [Kerbal Space Program](https://wiki.kerbalspaceprogram.com/wiki/Key_bindings), a
popular space simulation game that features complex orbital mechanics and spacecraft controls (I highly recommend the
original game). These shortcuts are designed to provide quick access to common functions and enhance the user experience
by reducing reliance on the mouse.

## System and UI Controls

| Shortcut | Function |
|----------|----------|
| `Shift + F1` | Toggle help overlay (side menu must be open) |
| `Shift + F2` | Toggle UI visibility |

## View Controls

| Shortcut | Function |
|----------|----------|
| `Numpad +` | Zoom in |
| `Numpad -` | Zoom out |
| `` ` `` (backtick) | Reset camera view |
| `V` | Cycle through camera modes (Earth-centered, free camera, etc) |
| `F11` | Toggle fullscreen mode |

### Earth Centered Mode

Earth-centered mode keeps the camera focused on Earth while allowing rotation around it.

| Shortcut | Function |
|----------|----------|
| `↑` | Pan view Up |
| `↓` | Pan view Down |
| `←` | Pan view Left |
| `→` | Pan view Right |
| `Shift` | Slow movement speed while dragging |
| `R` | Auto-rotate around Earth |
| `Home` | Focus view on Current Sensor |

### Free Camera Mode

Free camera mode allows users to move the camera freely in 3D space using traditional FPS controls.

| Shortcut | Function |
|----------|----------|
| `W` | Move forward |
| `A` | Move left |
| `S` | Move backward |
| `D` | Move right |
| `Q` | Move up |
| `E` | Move down |
| `Numpad8` | Rotate up |
| `Numpad2` | Rotate down |
| `Numpad4` | Rotate left |
| `Numpad6` | Rotate right |
| `Shift` | Slow movement speed |
